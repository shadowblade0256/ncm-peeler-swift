# ncm-peeler-swift
## ncm 格式削皮器

+ 把 ncm 格式（网易云加密音乐格式）文件转化为 mp3 或 flac 格式文件。

+ 保留标题、专辑、歌手、封面等元数据。*

> 附注：网易云音乐新版 macOS 客户端在下载 FLAC 加密格式时，有一定概率不写入元数据信息。
